# Grading Workflow & Organization

## How to use this directory
1. Create a new subfolder for each assignment (e.g., `assignment_1_intro`).
2. Drop the students' `.sql` submission files directly into that assignment's folder.
3. Ask me to review the assignments in that folder.
4. I will generate individual feedback files (e.g., `.md`, `.txt`, or `.pdf`) and place them in a `feedback` subfolder within that assignment folder.
5. You can then drag and drop the `feedback` subfolder into Google Drive to generate shareable links for your Google Sheets tracker.
